COMBAT SYSTEM SPEC — An Untitled Feesh
================================================================================


================================================================================
CORE FLOW
================================================================================

    1. Player selects 3 fish as tribute (fighter select screen)
    2. Cutscene: guard leads player to arena, boss appears, banter dialogue
    3. Player chooses which of the 3 fish to start with
    4. Twin-stick bullet hell: WASD moves, mouse aims, click fires
    5. On fish death: dramatic animation → boss pauses → remaining bullets
       clear → selection UI → new fish spawns at start position → boss resumes
    6. All 3 die: player expelled, all 3 fish lost (must re-catch)
    7. Boss defeated: surviving fish run off (also lost), dead fish left behind.
       All 3 are consumed regardless of outcome

================================================================================
DAMAGE FORMULAS
================================================================================

Player → Boss:
    damage_per_hit = fish_attack × weapon_modifier

    No boss "defense" stat — boss HP is the balancing lever. Some bosses have
    armored phases where damage is halved (broken by armor_break or completing
    a mechanic).

Boss → Player:
    effective_damage = bullet_base_damage × (100 / (100 + fish_defense))
    minimum damage = 1

    This is percentage-based reduction — defense never makes you immune, but
    high defense meaningfully reduces incoming damage.

i-frames:
    1.0 second of invulnerability after taking a hit (industry standard).
    Fish flashes during i-frames.


================================================================================
WEAPON CLASSES
================================================================================

Every fish has a dedicated weapon mapped to one of 11 weapon classes. Each
class defines the attack behavior, range, speed, and damage modifier. Higher
modifiers mean fewer attacks per second (and vice versa), so DPS roughly
normalizes — but melee classes have ~30-50% higher DPS than ranged to
compensate for risk.

Reference DPS (attack stat 100, per class):

    ─────────────   ─────────────────────────────────   ─────────────────────────   ─────────   ────────    ────
    SLASH           Arc swing toward cursor              Short (50px), 90° arc       2.5/sec     1.0x        250
    HEAVY_SLASH     Wide devastating swing               Med (65px), 140° arc        1.5/sec     1.5x        225
    THRUST          Straight-line lunge                  Med-Long (80–100px)         2.0/sec     1.1x        220

    WHIP            Medium-range thin sweep              Med (90px), 30° thin arc    2.0/sec     0.9x        180

RANGED CLASSES
    Class           Behavior                                        Range                       Speed           Modifier        ~DPS
    ─────────────   ─────────────────────────────────────────────   ─────────────────────────   ─────────────   ─────────────   ────
    SINGLE_SHOT     Hold to charge, release to fire                 Full arena                  1.2/sec         1.8x            216
    BOOMERANG       Thrown + returns, hits twice                     200px out+back              1.5/sec         0.6x per hit    180
    BLASTER         Standard aimed projectile                       Full arena                  2.5/sec         0.6x            150
    BEAM            Hold to fire continuous line (2s on/1s recharge) Long line                   8 ticks/sec     0.25x/tick      ~134
    RAPID_FIRE      Spray of weak projectiles                       Full arena                  9/sec           0.13x           117

Design intent:
    Melee fish are high-risk/high-reward — you deal more damage but must dodge
    bullet patterns point-blank. Ranged fish are safer but slower kills. This
    makes tribute selection meaningful: skilled players pick melee fish for fast
    clears, cautious players pick ranged for safety.

BOSS BULLET DAMAGE TIERS
================================================================================

    Bullet Type         Base Damage     Description
    ──────────────────  ─────────────   ─────────────────────────────────────────────────
    Small bullet        40–60           Frequent pattern filler
    Large bullet        80–120          Slow, telegraphed projectiles
    Boss slam/charge    150–250         Highly telegraphed, dodge or eat massive damage
    Stage hazard        40–80           Environmental ongoing damage (lava, ice patches)
    Phase 2+            All × 1.25–1.5  Damage scales with boss phase

Survivability math (hits to kill a fish from small bullets, base 50 damage):

    Fish Tier       Defense     Effective Damage    HP Range        Hits to Die
    ─────────────   ────────    ────────────────    ─────────────   ───────────
    Common          15          43                  260–360         6–8
    Uncommon        30          38                  360–600         9–16
    Rare            50          33                  650–900         20–27
    Legendary       80          28                  1,000–1,500     36–54

With 1-second i-frames, a Common survives minimum 6–8 seconds of continuous
hits. Realistically, getting hit every ~5 seconds by a dodging player, each
Common fish lasts 30–40 seconds. Three Commons = 1.5–2 minutes of total error
budget.

================================================================================
DESIGN DECISIONS (RESOLVED)
================================================================================

Ability Philosophy:
    Common fish = weapon class IS their identity. Simple, basic fighters.
    Specials concentrate on Legendaries, sprinkle into Rares, and a few
    Uncommons per biome get something minor. Not every fish needs a unique
    special — weapon class variety is enough for commons.

Melee HP Bonus:
    Melee classes (SLASH, HEAVY_SLASH, THRUST, SMASH, WHIP) get +15% HP bonus
    applied at combat load time. This compensates for the risk of closing
    distance in a bullet hell. Base data stays clean; bonus is a combat modifier.

        Tier            Base HP     With +15%   Extra Hits Survived (vs 50 base bullet)
        ─────────────   ────────    ─────────   ───────────────────────────────────────
        Common          300         345         +1
        Uncommon        480         552         +1–2
        Rare            750         863         +2–3
        Legendary       1,200       1,380       +3–4

Ability Distribution (per biome):
    Every biome has access to key ability categories. Abilities were
    redistributed to prevent biome-lock of important tools:

        Ability         Forest  Ocean   Jungle  FrozenMtn   Lava    Total
        ─────────────   ──────  ──────  ──────  ─────────   ──────  ─────
        stun            3       4       3       2           1       13
        bleed           2       1       1       1           1       6
        burn            0       0       0       0           4       4
        poison          1       0       0       0           0       1
        pierce          0       0       1       1           1       3
        multi_shot      2       3       3       2           0       10
        armor_break     1       0       0       1           1       3
        charge          1       2       2       1           0       6
        reflect         1       2       1       1           0       5
        heal            1       2       1       1           1       6
        berserk         2       0       0       1           0       3
        summon          0       2       0       0           0       2
        firestorm       0       0       0       0           1       1
        dash            0       0       0       0           1       1

    Changes made from original:
      - Sturgeon (Forest):        stun → armor_break (warhammer breaks armor)
      - Barracuda (Ocean):        null → bleed (dual daggers cause bleeding)
      - Discus (Jungle):          null → pierce (chakram cuts through)
      - Hatchetfish (Jungle):     null → bleed (hatchet strikes bleed)
      - Frostbite Eel (FrozenMtn): stun → bleed (whip lashes bleed)
      - Crystal Perch (FrozenMtn): null → pierce (crystal sword cuts through)
      - Snowball Sunfish (FrozenMtn): null → heal (heal option for FrozenMtn)
      - Avalanche Catfish (FrozenMtn): charge → armor_break (hammer smashes armor)
      - Ember Eel (Lava):         null → bleed (lash weapon, bleeds)
      - Firefly Fish (Lava):      null → heal (heal option for Lava)

Crits:
    No global crit system. A few specific weapons or fish may have crit as
    their unique shtick — to be determined on fish-by-fish basis if needed.

Balance Philosophy:
    Every boss is beatable with the worst fish in the game. Skill and pattern
    recognition are king. Better fish make it easier, not trivially so.
    Legendary fish = much easier time, but still a challenge. Not a faceroll.
